# Space Runner

A 2D platformer game developed using **Python** and the **Pygame** library.

## 🚀 Overview
The game follows the journey of a character navigating through five unique levels, each themed around the four seasons (Spring, Summer, Autumn, Winter) and a final challenging Boss stage. It features classic platformer mechanics, including character movement, projectile combat, and dynamic obstacle navigation.
